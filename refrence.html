BASIC STATEMENTS

println

Used to Display something on the screen. 

clear()
println "Hello World!"

Can also be used to declare variables!

clear()
input("Enter the value of a", a)
println a

clear()

Used to clear the screen. 

clear()
println("Hello World")
clear()
println("Hello World again!")

input()

Used to ask the user for some value

clear()
input("Enter your name: ", a$)
println a$

if

Used for boolean operations

clear()
input("Want to continue? (Y/N)", ch$)
if(ch$ = "y") {
	println "Thank you"
}

else

Used in if, for the false block

clear()
input("Want to continue? (Y/N)", ch$)
if(ch$ = "y") {
	println "Thank you"
else {
	println "You suck!"
}
}

elif

Used in if, for other blocks

clear()
input("Want to continue? (Y/N)", ch$)
if(ch$ = "y") {
	println "Thank you"
elif(ch$ = "n") {
	println "You suck!"
else {
	println "Invalid Choice!"
}
}
}

for

Used to loop a certain block of statement/s for a certain amount of time.

clear()
for i = 1 to 10 {
	println i
}

while

Used to loop a certain block of statement/s.

clear()
while(x < 10) {
	println x
	x=x+1
}

setScreen

PL.QB

Used to set the screen from anywhere in number 0 - 13.

clear()
setScreen 12
line(100,100)-(200,200), 5

PL.QB64

Used to set screen to a certain resolution

clear()
setScreen(800,600)
line(0,0)-(800,600), rgb(100,200,255)

line

Used to join any two points.

clear()
setScreen 12
line(0,0)-(100,200)

ellipse

Used to draw a circle with a certain radius.

clear()
setScreen 12
ellipse(150,150), 50

funct

Used to Create a function.

clear()
input("Enter a", a)
input("Enter b", b)
sum = add(a,b)
println "The Sum of those number = "; sum

funct add(num1,num2) {
	add = num1 + num2
}

void

Used to create a empty function(sub)

clear()
display_name

void display_name() {
	println "Prithak Adhikari"
}

createVector

Used to create a vector.

clear()
createVector vector
vector.x = 100
vector.y = 200
println vector.x ; vector.y

declare

Used to declare something to something.

declare a[10]
for i = 1 to 10 {
	a[i] = i
}
